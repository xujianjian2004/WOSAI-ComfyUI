# WOSAI-ComfyUI

WOSAI 是专业的可视化 ComfyUI 节点美化与增强工具，好看更强悍！WOSAI is a professional visualization toolkit designed to enhance the functionality and aesthetic presentation of ComfyUI nodes.

| 节点 | 分类 | 说明 |
| --- | --- | --- |
| **画布注释 CanvasNote** | 工具 | 画布注释节点，支持丰富样式、Markdown、内置/自定义预设 |

## 画布注释 Canvas Note

一个轻量级的 ComfyUI 画布注释节点，具有丰富的样式、内置预设和自定义设置功能，可直接在工作流画布上添加标题、备注与说明。

### 核心特性

- **零端口**：无输入/输出，纯装饰与信息注释节点
- **支持透明**：背景透明度为 0 时，与任何画布主题无缝融合
- **原生文本编辑**：支持中英文混排、IME 输入法、复制粘贴、文本选中
- **随画布缩放**：文字字号与内边距随画布缩放比例等比同步
- **双向自适应**：输入/删除文字，调整字号、行距或切换预设时，节点高度实时撑开或收缩
- **边框与阴影联动**：共用一个拾色器，颜色同步
- **样式完整序列化**：所有样式属性随工作流 JSON 保存与加载
- **自定义预设持久化**：自定义预设存储于浏览器本地
- **Markdown 支持**：粗体、斜体、删除线、代码、链接、图片、标题、列表等格式
- **智能换行**：CJK/英文混排自动换行 + 行首禁排规则

### 基本操作

| 操作 | 效果 |
| --- | --- |
| 双击节点 | 编辑内容，并打开样式编辑器 |
| 拖拽边缘 | 手动调整节点宽度与高度 |

### 样式编辑器

双击节点文字区域即可打开，弹出位置默认位于节点左侧或右侧，自动防止超出屏幕边界。

**内置预设：**

| 预设 | 风格 | 字号 |
| --- | --- | --- |
| 标题 | 蓝色背景，白色加粗文字，圆角边框 | 72px |
| 序号 | 窄条红棕编号背景，白色粗体数字，正圆造型 | 60px |
| 正文 | 绿色背景，白色普通文字，带边框 | 24px |
| 便签 | 黄色便签风格，深褐色文字，圆角+阴影 | 24px |
| 注释 | 无色透明背景，白色普通文字，无边框 | 52px |

**样式参数：**

| 参数 | 范围 | 说明 |
| --- | --- | --- |
| 大小 | 8–200 px | 文字字号 |
| 行距 | 0.8–3.0 | 行间距倍数 |
| 透明 | 0–100% | 背景填充透明度 |
| 边框 | 0–10 px | 0 px 即无边框 |
| 阴影 | 0–60 px | 发光 / 阴影扩散半径 |
| 圆角 | 0–300 px | 节点与边框的圆角半径 |
| 内边距 | 0–60 px | 文字与节点边缘的距离 |

## 尺寸选择 SizeSelect

一个轻量、直观的尺寸选择节点，支持预设分辨率快速选择与自定义宽高输入。

### 功能特性

- 四档预设分辨率（SD / HD / FHD / QHD）× 八种宽高比
- 自定义宽高输入（256–2048 px），自动向下对齐 8 的倍数
- 一键互换宽高数值
- 实时尺寸预览，切换即刷新

### 预设尺寸对照表

| 宽高比 | SD (0.2–0.4MP) | HD (0.6–1.0MP) | FHD (1.1–2.8MP) | QHD (2.4–5.0MP) |
| --- | --- | --- | --- | --- |
| 3:2 | 768×512 | 1152×768 | 1536×1024 | 2304×1536 |
| 2:3 | 512×768 | 768×1152 | 1024×1536 | 1536×2304 |
| 4:3 | 512×384 | 1024×768 | 1280×960 | 2048×1536 |
| 3:4 | 384×512 | 768×1024 | 960×1280 | 1536×2048 |
| 16:9 | 640×360 | 1280×720 | 1920×1080 | 2560×1440 |
| 9:16 | 360×640 | 720×1280 | 1080×1920 | 1440×2560 |
| 21:9 | 768×328 | 1280×544 | 2560×1080 | 3440×1440 |
| 1:1 | 512×512 | 768×768 | 1024×1024 | 1536×1536 |

### 输入/输出

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| Manual_Mode | COMBO | `off` = 预设模式，`on` = 自定义模式 |
| Resolution | COMBO | SD / HD / FHD / QHD（预设模式） |
| Aspect_Ratio | COMBO | 8 种宽高比（预设模式） |
| Custom_Width | INT | 256–2048 px，步长 8（自定义模式） |
| Custom_Height | INT | 256–2048 px，步长 8（自定义模式） |
| **width** (出) | INT | 目标宽度 |
| **height** (出) | INT | 目标高度 |

## 万能滑条 OmniSlider

多通道独立滑条控制节点，支持浮点/整数双类型与多种视觉样式。

### 核心特性

- **1~6 独立通道**：+/- 按钮动态增减通道，每个通道独立输出各自的值，全通道同时激活
- **双输出类型**：每个通道可独立切换浮点数（FLOAT）或整数（INT）
- **双视觉样式**：进度滑条（传统条形）+ 温度滑条（填充样式，左右色温对比）
- **独立配色**：每个通道独立色环，温度滑条模式可分别设置左/右/按钮/数字颜色
- **缩放比例**：统一调节所有滑条轨道高度

### 基本操作

| 操作 | 效果 |
| --- | --- |
| 拖动滑条 | 调节当前通道的输出值 |
| 双击滑条轨道 | 打开设置面板 |
| 右键节点 → 菜单项 | 打开设置面板 |
| 方向键 ↑ ↓ | 按步长微调值 |
| Home / End | 跳转到最小值 / 最大值 |

### 设置面板

| 参数 | 说明 |
| --- | --- |
| 显示名称 | 通道标签文字，显示于滑条前方与输出端口名 |
| 输出类型 | 浮点数（FLOAT）或整数（INT） |
| 最小值 / 最大值 | 滑条取值范围 |
| 步长 | 方向键调节的最小粒度 |
| 颜色 | 通道专属色环，温度滑条模式扩展为四色配置 |

### 输入/输出

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| active_value | FLOAT | 内部缓存键，自动同步滑条值（勿手动修改） |
| **滑条1 ~ 滑条6** (出) | FLOAT | 各通道独立输出值，未启用通道输出 0.0 |

## 节点配色 NodeColor

对任意 ComfyUI 节点生效的颜色扩展，支持纯色/多色渐变背景和标题文字样式。

### 核心特性

- **全局适用**：对任意节点生效，不影响节点原有功能
- **1/2/3 色渐变**：纯色、双色、三色一键切换；纯色模式下标题背景 / 面板背景 / 整体背景三目标独立编辑
- **12 组内置预设**：纯色 12 色 + 渐变 12 组；按住 **Shift** 切换 12 级灰度色卡
- **渐变方向**：8 方向（上下左右 + 四对角）
- **标题样式**：独立调节节点标题的字号、对齐、字重（经典 Canvas 与 Nodes 2.0 双渲染路径）
- **取色工具**：HEX 输入框（单击全选复制）、屏幕拾色器（EyeDropper）、随机色（Alt+点击 = 多选节点各配随机色）
- **取色历史与自定义预设**：localStorage + 服务端 JSON 双层持久化（`presets/color_presets.json`），跨浏览器共享
- **分组配色**：分组右键独立取色器
- **完整序列化**：渐变与标题样式随工作流 JSON 保存/加载

### 基本操作

| 操作 | 效果 |
| --- | --- |
| 右键节点 → 🟠 节点配色 | 打开配色面板 |
| 快捷键 C | 选中节点后按 C 打开配色面板 |
| 单色/双色/三色按钮 | 纯色 ↔ 双色 ↔ 三色切换 |
| 点击预设色块 | 一键应用预设配色 |
| 按住 Shift | 纯色预设切换为灰度色卡 |

## 快捷上色条 ColorBar

常驻画布的快捷上色 HUD，高频上色无需打开完整面板。

### 核心特性

- **悬浮球**：常驻画布的 🎨 圆球，可拖拽到任意位置（位置记忆），点击滑动展开/收起工具条
- **横版 / 竖版**：一键切换布局并记忆，工具条与配色菜单形态联动
- **6 精选预设色**：亮（橙/金）、中（绿/蓝）、暗（紫/炭黑），点击即对选中节点/分组上色
- **拾色器 + 取色历史**：屏幕任意位置取色自动应用，保留最近 3 条历史（悬停 × 可删除）
- **按节点类型一键上色**：节点自动归入 12 个功能大类（输入/引导/采样/输出/解码/编码/模型/提示/图像/视频/音频/工具），12 组配色主题一键全图上色，支持撤销
  - 纯色 Solid：柔和 / 鲜明 / 暗调 / 粉黛 / 森系 / 海雾
  - 渐变 Gradient：流岚 / 幻彩 / 深邃 / 暮山 / 碧波 / 翠谷
- **随机 / 清除**：随机色（Alt = 多节点各配不同色）、一键清除颜色
- **分组支持**：所有操作对选中的分组（Group）同样生效
- **Hold 模式**（设置中开启）：按住 `` ` `` 显示工具条，悬停某项松开按键即应用，全程免点击

### 基本操作

| 操作 | 效果 |
| --- | --- |
| 点击悬浮球 / 快捷键 `` ` `` | 展开 / 收起工具条 |
| 拖动悬浮球 | 移动位置（自动记忆） |
| 预设 | 打开"按节点类型一键上色"菜单，连续试色不关窗 |
| 高级 | 打开完整 NodeColor 调色面板 |
| ⇅ / ⇄ | 切换横版 / 竖版 |

### 设置项（ComfyUI 设置 → 搜索 ColorBar）

| 设置 | 默认 | 说明 |
| --- | --- | --- |
| 显示悬浮球 | 开 | 关闭后仍可用快捷键唤起 |
| Hold 模式 | 关 | 按住 `` ` `` 显示，悬停松开即应用 |

## 技术说明

- 遵循 [ComfyUI 自定义节点规范](https://docs.comfy.org/zh-CN/custom-nodes/overview)
- Node Note 纯前端渲染，Python 仅负责节点注册
- Size Select 依赖 ComfyUI app.js 模块，需 ComfyUI 0.2.0+
- NodeColor 渐变与标题样式同时支持经典 Canvas 与 Nodes 2.0（Vue DOM）双渲染路径
- 取色历史/自定义预设双层持久化：localStorage 即时读写 + 插件目录 `presets/color_presets.json`（服务端不可用时自动降级）
- 屏幕拾色器使用原生 EyeDropper API，需 Chrome/Edge 95+（不支持时按钮自动禁用）
- 其余自定义预设（注释样式等）存储于浏览器 localStorage
