WOSAI 是专业的可视化 ComfyUI 节点美化与增强工具，好看更强悍！WOSAI is a professional visualization toolkit designed to enhance the functionality and aesthetic presentation of ComfyUI nodes.

## 画布注释 Canvas Note

一个轻量级的 ComfyUI 画布注释节点，具有丰富的样式、内置预设和自定义设置功能，可直接在工作流画布上添加标题、备注与说明。

### 核心特性

- **零端口**：无输入/输出，纯装饰与信息注释节点
- **支持透明**：背景透明度为 0 时，与任何画布主题无缝融合
- **原生文本编辑**：支持中英文混排、IME 输入法、复制粘贴、文本选中
- **随画布缩放**：文字字号与内边距随画布缩放比例等比同步
- **双向自适应**：输入/删除文字，调整字号、行距或切换预设时，节点高度实时撑开或收缩
- **边框与阴影联动**：共用一个拾色器，颜色同步
- **样式完整序列化**：所有样式属性随工作流 JSON 保存与加载
- **自定义预设持久化**：自定义预设存储于浏览器本地
- **Markdown 支持**：粗体、斜体、删除线、代码、链接、图片、标题、列表等格式
- **智能换行**：CJK/英文混排自动换行 + 行首禁排规则

### 基本操作

| 操作 | 效果 |
| --- | --- |
| 双击节点 | 编辑内容，并打开样式编辑器 |
| 拖拽边缘 | 手动调整节点宽度与高度 |

### 样式编辑器

双击节点文字区域即可打开，弹出位置默认位于节点左侧或右侧，自动防止超出屏幕边界。

**内置预设：**

| 预设 | 风格 | 字号 |
| --- | --- | --- |
| 标题 | 蓝色背景，白色加粗文字，圆角边框 | 72px |
| 序号 | 窄条红棕编号背景，白色粗体数字，正圆造型 | 60px |
| 正文 | 绿色背景，白色普通文字，带边框 | 24px |
| 便签 | 黄色便签风格，深褐色文字，圆角+阴影 | 24px |
| 注释 | 无色透明背景，白色普通文字，无边框 | 52px |

**样式参数：**

| 参数 | 范围 | 说明 |
| --- | --- | --- |
| 大小 | 8–200 px | 文字字号 |
| 行距 | 0.8–3.0 | 行间距倍数 |
| 透明 | 0–100% | 背景填充透明度 |
| 边框 | 0–10 px | 0 px 即无边框 |
| 阴影 | 0–60 px | 发光 / 阴影扩散半径 |
| 圆角 | 0–300 px | 节点与边框的圆角半径 |
| 内边距 | 0–60 px | 文字与节点边缘的距离 |

## 尺寸选择 SizeSelect

尺寸选择与控制节点，支持预设分辨率快速选择、自定义宽高输入、以及图像/遮罩/latent 缩放输出。

### 功能特性

- **双模式**：预设模式（四档分辨率 × 八种宽高比）+ 自定义模式（256–2048 px，自动 8 倍对齐）
- **一键互换宽高**：竖排按钮点击/拖拽，宽高数值瞬间交换
- **实时尺寸预览**：当前选中/自定义的宽高数值在节点下方实时显示
- **图像/遮罩处理**：连接图像或遮罩输入时，自动中心裁剪并缩放至目标尺寸输出
- **Latent 处理**：连接 latent 输入时，自动缩放至目标尺寸输出（含 `downscale_ratio_spacial` 字段）
- **VAE 编码**：连接 VAE 输入时，缩放后的图像通过 VAE 编码为真实 latent（非零张量）
- **无输入兼容**：未连接任何输入时，输出尺寸值 + 空 latent（向后兼容）

### 端口布局

```
左侧输入（上→下）           右侧输出（上→下）
┌──────────────┐           ┌──────────────┐
│ image  (图像) │ ←── 对应 ──→ │ 图像  (IMAGE) │
│ mask   (遮罩) │ ←── 对应 ──→ │ 遮罩  (MASK)  │
│ latent        │ ←── 对应 ──→ │ latent        │
│ vae           │           │ 宽度  (INT)    │
│              │           │ 高度  (INT)    │
└──────────────┘           └──────────────┘
```

### 处理优先级

| 优先级 | 输入情况 | 输出结果 |
| --- | --- | --- |
| 1 | 连接了 image/mask | 图像中心裁剪 + LANCZOS 缩放 → image / mask / latent（如有 VAE 则真实编码） |
| 2 | 仅连接了 latent | latent 中心裁剪 + interpolate 缩放 → latent |
| 3 | 无任何连接 | 输出尺寸值 + 空 latent（与原行为完全兼容） |

| 3:2 | 768×512 | 1152×768 | 1536×1024 | 2304×1536 |
| 2:3 | 512×768 | 768×1152 | 1024×1536 | 1536×2304 |

| 3:4 | 384×512 | 768×1024 | 960×1280 | 1536×2048 |

### 基本操作

| 操作 | 效果 |
| --- | --- |
| 点击预设按钮（SD / HD / FHD / QHD） | 选择分辨率档位 |
| 点击宽高比按钮（9 个竖排） | 切换宽高比 |
| 切换模式开关（预设 ⇄ 手动） | 预设模式与自定义宽高输入切换 |
| 点击一键互换宽高 | 宽高数值瞬间交换 |
| 拖拽水平条 | 实时调节自定义宽高（自定义模式） |

## 万能滑条 OmniSlider

多通道独立滑条控制节点，支持浮点/整数双类型、双视觉样式与实时设置面板。

### 核心特性

- **多通道独立**：滑条总数上限可设（1~6），通过设置面板按钮任意增减；每个通道独立输出，全通道同时激活；增减时输出端口数同步匹配
- **双输出类型**：每个通道可独立切换浮点数（FLOAT）或整数（INT）
- **双视觉样式**：一体式（传统条形）+ 圆点式（填充样式，左右色温对比）
- **独立配色**：每个通道独立色环，圆点式可分别设置左/右/按钮/数字颜色
- **端口自适应**：连到下游输入时自动识别端口名与类型，也可自定义滑条名
- **滚轮调节**：滑条、面板缩放、配色色相条均支持滚轮上下快速增减
- **防误触**：多条上锁（只读）+ 单击不跳值（仅拖动越过阈值才改值）
- **极简模式**：可独立勾选隐藏「标题面板 / 角标 / 端口号」，经典 Canvas 与 Nodes 2.0（Vue DOM）双渲染均生效
- **实时设置面板**：右键滑条打开；所有调整即时预览、关闭即生效（无确认按钮）

### 基本操作

| 操作 | 效果 |
| --- | --- |
| 拖动滑条 | 调节当前通道的值（极简模式下单击不跳值，需拖动） |
| 滚轮 | 按步长增 / 减当前滑条的值 |
| 右键滑条 | 打开实时设置面板 |
| 右键节点 → 🟠 万能滑条 OmniSlider | 打开设置面板（兜底入口） |
| 点击锁图标 | 上锁 / 解锁，锁定后只读防误触 |
| 方向键 ↑ ↓ / Home / End | 按步长微调 / 跳到最小 / 最大值 |

### 设置面板（分组，关闭即生效）

| 分组 | 内容 |
| --- | --- |
| 顶部 | 选择滑条总数（1~6 数字按钮）· 通道标签 C1…CN（带三角指针）· 修改滑条名称输入框 |
| 滑条参数类型 | 浮点 / 整数（按钮点选）· 范围（最小值 ~ 最大值 · 步长） |
| 样式 + 缩放 + 颜色 | 一体式 / 圆点式样式切换 · 缩放比例（滚轮可调）· 取色器与预设色点（圆点式扩展四色标签） |
| 隐藏节点 | 标题面板 / 角标 / 端口号 三个独立多选按钮 |

### 输入/输出

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| active_value | FLOAT | 内部缓存键，自动同步滑条值（勿手动修改） |
| **C1 ~ CN** (出) | FLOAT / INT | 各通道独立输出，端口名随连接自动识别或自定义；INT 通道输出整数 |

## 高级配色 NodeColor

对任意 ComfyUI 节点生效的颜色扩展，支持纯色/多色渐变背景和标题文字样式。

### 核心特性

- **全局适用**：对任意节点生效，不影响节点原有功能
- **1/2/3 色渐变**：纯色、双色、三色一键切换；纯色模式下标题背景 / 面板背景 / 整体背景三目标独立编辑
- **12 组内置预设**：纯色 6色 + 渐变 6 组；按住 **Shift** 切换 12 级灰度色卡
- **渐变方向**：8 方向（上下左右 + 四对角）
- **标题样式**：独立调节节点标题的字号、对齐、字重（经典 Canvas 与 Nodes 2.0 双渲染路径）
- **取色工具**：HEX 输入框（单击全选复制）、屏幕拾色器（EyeDropper）、随机色（Alt+点击 = 多选节点各配随机色）
- **取色历史与自定义预设**：localStorage + 服务端 JSON 双层持久化（`presets/color_presets.json`），跨浏览器共享
- **分组配色**：分组右键独立取色器
- **完整序列化**：渐变与标题样式随工作流 JSON 保存/加载

### 基本操作

| 操作 | 效果 |
| --- | --- |
| 右键节点 → 🟠 高级配色 NodeColor | 打开配色面板 |
| 单色/双色/三色按钮 | 纯色 ↔ 双色 ↔ 三色切换 |
| 点击预设色块 | 一键应用预设配色 |
| 按住 Shift | 纯色预设切换为灰度色卡 |

## 快捷上色条 ColorBar

常驻画布的快捷上色 HUD，高频上色无需打开完整面板。

### 核心特性

- **悬浮球**：常驻画布的全彩渐变圆球，可拖拽到任意位置（位置记忆），点击滑动展开/收起工具条
- **横版 / 竖版**：一键切换布局并记忆，工具条与配色菜单形态联动
- **预设色（4 组 × 6 色）**：经典 / 浓郁 / 柔和 / 鲜亮 四组配色，点击「换一组预设」刷新图标循环切换（中心显示当前组号、记忆所选组），点击色点即对选中节点/分组上色
- **拾色器 + 取色历史**：屏幕任意位置取色自动应用，保留最近 3 条历史（悬停可删除）
- **按节点类型一键上色**：节点自动归入 12 个功能大类，12 组配色主题一键全图上色，可撤销
- **随机 / 清除**：随机色（Alt = 多节点各配不同色）、一键清除颜色
- **分组支持**：所有操作对选中的分组（Group）同样生效
- **Hold 模式**（设置中开启）：按住 `` ` `` 显示工具条，悬停某项松开按键即应用，全程免点击

### 基本操作

| 操作 | 效果 |
| --- | --- |
| 点击悬浮球 / 快捷键 `` ` `` | 展开 / 收起工具条 |
| 拖动悬浮球 | 移动位置（自动记忆） |
| 预设 | 打开"按节点类型一键上色"菜单，连续试色不关窗 |
| 高级 | 打开完整 NodeColor 调色面板 |
| ⇅ / ⇄ | 切换横版 / 竖版 |

### 设置项（ComfyUI 设置 → 搜索 ColorBar）

| 设置 | 默认 | 说明 |
| --- | --- | --- |
| 显示悬浮球 | 开 | 关闭后仍可用快捷键唤起 |
| Hold 模式 | 关 | 按住 `` ` `` 显示，悬停松开即应用 |

## 技术说明

- 遵循 [ComfyUI 自定义节点规范](https://docs.comfy.org/zh-CN/custom-nodes/overview)
- Node Note 纯前端渲染，Python 仅负责节点注册
- 前端模块采用 ES module 架构，`lib/` 目录为共享模块，通过 `extension.json` 加载入口文件
- Size Select 依赖 ComfyUI 0.2.0+，图像处理需要 PIL/Pillow + numpy（已在 requirements.txt 中声明）
- NodeColor 渐变与标题样式同时支持经典 Canvas 与 Nodes 2.0（Vue DOM）双渲染路径
- OmniSlider 隐藏模式通过 CSS 注入实现 Nodes 2.0 兼容（非 MutationObserver 反复打补丁）
- 取色历史/自定义预设双层持久化：localStorage 即时读写 + 插件目录 `presets/color_presets.json`（服务端不可用时自动降级）
- 屏幕拾色器使用原生 EyeDropper API，需 Chrome/Edge 95+（不支持时按钮自动禁用）
- 统一 tooltip 机制：所有面板使用 `lib/tooltip.js` 共享实现，零延迟即时显示，视口自动钳制
